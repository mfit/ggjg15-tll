Install + Build
================

Prerequisites
-------------

Install npm / node (javascript package manager)

http://nodejs.org/

e.g. $ sudo apt-get install node

Install grunt
http://gruntjs.com/installing-grunt

(Javascript task runner)


Install
-------

Checkout the source

$ npm install
$ bower install


Run development server
-----------------------

$ grunt serve
